# CLI System Assistant

Простая и удобная консольная утилита на Python для мониторинга состояния операционной системы и быстрого управления запущенными процессами.

## Функционал

- 📊 **Мониторинг системы**: вывод текущей загрузки процессора, использования оперативной памяти и свободного места на диске C:.
- 🎯 **Управление процессами**: быстрый поиск и принудительное завершение (kill) процессов по их имени.
- ⏳ **Реалистичные задержки**: имитация загрузки данных с помощью случайных тайм-аутов для более плавного UX.
- 🛡️ **Безопасность**: корректная обработка системных ошибок доступа (`AccessDenied`) и прерывания программы пользователем (`Ctrl+C`).

## Требования

Для работы скрипта необходим **Python 3.6+** и библиотека `psutil`.

```bash
pip install psutil
```

## Использование

1. Клонируйте репозиторий или скачайте файл со скриптом.
2. Запустите скрипт через терминал:

```bash
python main.py
```

### Доступные команды

После запуска введите одну из следующих команд в консоль:

- `система` — показать информацию о нагрузке на ЦП, ОЗУ и диск.
- `процесс` — открыть меню для закрытия процесса (например, `chrome.exe`).
- `помощь` — вывести список всех доступных команд.
- `автор` — посмотреть информацию о создателе проекта.
- `выход` — завершить работу утилиты.
